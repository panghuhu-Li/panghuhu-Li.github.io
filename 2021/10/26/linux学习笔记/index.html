Linux学习笔记

vim常用指令

  • yy 复制一行 5yy复制5行以此类推
  • dd 删除一行 5dd删除5行以此类推
  • p粘贴
  • G尾行 gg首行
  • :set nu set nonu 显示行号/不显示行号

开关机指令

  • shutdown -h 1 一分钟后关机
  • shutdown -r now 立即重起
  • reboot 重起
  • halt 关机
  • sync 将内存数据保存到磁盘
  • logout 注销（f非图形界面）
  • su 登录超级管理员账户（root）

用户管理

  • 添加用户

    useradd 用户名 默认用户的家目录在/home中

    useradd -d 指定新用户路径

  • 用户密码修改

    passwd 用户名 （pwd显示当前目录）

  • 用户删除

    userdel 用户名（保留家目录）

    userdel -r 用户名（删除对应家目录）

  • 查询用户信息

    id 用户名

  • 切换用户

    su -用户名 （权限高用户切换到权限低用户时不需要输入密码）

  • 查看当前用户

    whoami /who am i(登录的用户信息)

  • 用户组

    具有相同的属性/权限

    groupadd 组名

    groupdel 组名

    useradd -g wudang zwj(表示添加用户zwj到wudang组)

    usermod -g 用户组 用户名(修改用户到对应组)

指定运行级别

  • 0：关机
  • 1：单用户（找回丢失密码）
  • 2：多用户状态没有网络服务（很少使用）
  • 3：多用户状态有网络服务（经常使用）
  • 4：系统未使用保留给用户
  • 5：图形界面
  • 6：系统重启

常用级别是3和5

init [0123456] 切换不同的运行级别

systemctl get-default 查看当前级别

systemlctl set-default 设置当前级别

文件目录指令

  1. cd ~ 回到家目录

    cd 绝对路径和相对路径

    cd .. 回到当前目录的上一级目录

  2. mkdir [选项] 要创建的目录 用于创建目录

    -p 表示创建多级目录

  3. rmdir [选项] 要删除的目录 用于删除目录

    删除的目录必须是空目录，如果删除非空目录需要使用rm -rf

  4. cp [选项] source dest source:表示源文件 dest:目的路径

    -r :递归复制整个文件夹

    \cp 强制覆盖不提示

  5. rm [选项] 要删除的文件或目录

    -r : 递归的删除整个文件夹

    -f : 强制删除不提示

  6. mv oldNameFile newNameFile(source dest) 移动文件目录或重命名

  7. cat [选项] 要查看的文件

    -n ：显示行号

    为了浏览方便一般加上 管道命令|more

  8. less 用来分屏查看，每次只将整个文件加载之后，适合用来查看大文件

    空格键：向下翻动一页；

    pagedown：向下翻动一页

    pageup：向上翻动一页

    /字串 向下搜寻“字串”的功能；n：向下查找，N：向上查找（掌握一个即可 ）

    ?字串 向上搜寻“字串”的功能；n：向上查找，N：向下查找

    q：离开这个程序

  9. echo [选项] [输出内容] 输出内容到控制台

  10. head 用于显示文件的开头内容，默认情况下显示文件前10内容

    head -n 5 文件 表示显示该文件前5行内容

  11. tail 用于显示文件尾部的内容，默认情况下显示文件后10行的内容

    -n 用来选择显示的行数

  12. ln 软连接，相当于windows中的快捷方式

    ln -s [源文件目录] [软连接名]

    eg: ln -s /root/ /home/myroot 相当于在home建立一个快捷方式myroot指向root

    rm /home/myroot 删除该快捷方式

  13. history 查看所有的曾经执行的命令

    history 10表示执行最近使用过10个指令

时间日期类

  1. date 显示当前时间
  2. date +%Y 显示当前的年份
  3. date “+%Y-%m-%d” 显示年月日
  4. date +%m 显示月份
  5. date -s 字符串时间 设置系统的时间
  6. cal 显示当前日历
  7. cal 2020 显示2020年所有日历

查找指令

  1. find [搜索范围] [选项] 文件

    选项：-name 按照文件名进行查找

    ​ -user 查找属于指定用户的文件

    ​ -size 按照文件大小进行查找

  2. locate 快速定位文件路径，无需遍历整个文件系统

    在执行之前必须先要执行updatedb

    which 可以查看某个指令在哪个目录下

  3. grep 过滤查找 管道符“|” 表示将前一个命令的处理结果输出传递给后面的命令处理

    eg: cat a.txt | grep “hello” 表示将a.txt中的内容交给grep 并过滤出只有hello的内容

压缩和解压

  1. gzip 用于压缩文件 只能将文件压缩为.gz文件

    gunzip用于解压文件

  2. zip [选项] XXX.zip 将要压缩的内容

    unzip [选项] XXX.zip 解压缩文件

    -r 递归压缩，即压缩目录

    -d<目录> 指定解压后文件存放的目录

  3. tar 既可以解压也可以压缩（既可以是文件也可以是文件夹）压缩后为.tar.gz文件

    tar -zcvf 压缩的文件名 文件[1,2,3…] 将文件[1,2,3…]压缩为对应文件名

    tar -zxvf 要解压的文件名 -C要解压到的位置

组管理和权限管理

  1. ls -ahl 查看文件的所有者

    chown 用户名 文件名 修改文件的所有者

  2. chgrp 组名 文件名 修改文件所在的组

  3. usermod -g 新组名 用户名 改变用户所在的组

  4. usermod -d 目录名 用户名 改变该用户登录的初始目录。用户需要有进入该目录的权限

  5. ls -l 显示内容

    第0位确定文件类型

    ​ l表示链接，相当于windows的快捷方式

    ​ d是目录，相当于windows的文件夹

    ​ c是字符设备文件，鼠标，键盘

    ​ b是块设备，如：硬盘

    ​ -表示普通文件，如 a.txt

    第1-3位确定所有者（该文件的所有者）拥有该文件的权限。–User

    第4-6位确定所属组（同用户组的）拥有该文件的权限 ，—Group

    第7-9位确定其他用户拥有该文件的权限 –Other

  6. chmod 修改文件或者目录的权限

    u：所有者 g:所有组 o:其他人 a:所有人

    r=4,w=2,x=1

    eg: chmod u=rwx,g=rx,o=x 文件/目录名

  7. chown newowner 文件/目录 改变所有者

    chown newowner:newgroup 文件/目录 改变所有者和所在组

    -R 如果是目录 则使其下所有子文件或目录递归生效
